## 1. Документы

### 1.1. Общее
В CRM заготовителя хранятся документы, связанные с закупкой, продажей и контрагентами.

Документы хранятся как файлы с привязкой к сущностям:
- к контрагенту (например, договор)
- к сделке закупки (например, закупочный акт, квитанция)
- к сделке продажи (например, ТТН, УПД, спецификация, договор, заявка на перевозку)

### 1.2. Типы документов первой версии
В первой версии поддерживаются следующие типы документов:
- договор с контрагентом
- закупочный акт
- квитанция
- ТТН (транспортная накладная)
- УПД
- спецификация
- счёт
- заявка на перевозку
- доверенность
- иной документ

Список типов документов первой версии зафиксирован настоящим разделом.

### 1.3. Работа с документами
По каждому документу фиксируется:
- тип
- наименование
- дата
- связанный объект (контрагент, сделка)
- файл
- автор загрузки
- комментарий

Документы загружаются вручную. Автогенерация документов на основе данных сделки не входит в первую версию.

Документы могут быть просмотрены, скачаны, заменены и удалены с соответствующим правом.

### 1.4. Поиск и фильтрация
Экран «Документы» предоставляет общий список с фильтрами:
- по типу
- по контрагенту
- по сделке
- по периоду
- поиск по названию

Экран доступен ролям в соответствии с ролевой моделью. Финансовые и расчётные документы доступны только собственнику и бухгалтеру.

### 1.5. Хранилище и технические правила работы с файлами

Файлы документов CRM хранятся в едином файловом сервисе платформы. Полные правила работы с ними — `07-platform/storage.md`, раздел 5.6 (документы CRM). В частности:

- уровень доступа — `company-private` (доступны только сотрудникам компании-владельца в соответствии с ролевой моделью CRM); финансовые и расчётные документы — только собственнику и бухгалтеру (см. раздел 1.4);
- максимальный размер одного документа — 25 МБ;
- допустимые форматы — PDF, DOCX, XLSX, ODT, ODS, JPG, PNG, WEBP, TXT, CSV;
- общее число документов на компанию, на сделку, на контрагента — без ограничения;
- все загружаемые файлы проходят обязательную антивирусную проверку (см. `07-platform/storage.md`, раздел 7);
- при удалении компании документы CRM удаляются (см. `07-platform/storage.md`, раздел 9.2);
- при архивации контрагента (`03-crm/contractors.md`, раздел 1.7) документы сохраняются в архивной карточке.

Архитектурные детали (конкретный S3-провайдер, антивирусный движок, подписанные ссылки) — `07-platform/storage.md` и `08-architecture/`.
