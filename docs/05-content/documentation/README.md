Документ ещё не утверждён. Раздел будет зафиксирован отдельно.

Архитектурно модуль «База документации» предполагается реализовать на той же основе, что и модуль «База знаний» (`05-content/knowledge-base/`): дерево статей до 3 уровней, блочный редактор, двухстадийный workflow `draft` → `published`, доступ при любой подписке.

Содержательно модуль будет посвящён шаблонам документов, инструкциям и регламентам. Конкретные специализированные блоки (например, шаги инструкции, нумерованные регламенты) и категоризация дерева — открытый вопрос на момент детальной проработки.

В первой версии (MVP) модуль не реализуется.

См. `05-content/knowledge-base/other-integrations.md`, раздел 7 — связь с базой знаний и архитектурное переиспользование.
