## 1. Единая модель рейтингов

Рейтинг компаний на платформе строится по единой модели:
- если отзывов нет, в интерфейсе отображается «Рейтинг отсутствует»
- стартовое значение пять звёзд используется только как служебная база для первого расчёта
- до появления первого отзыва эти пять звёзд не отображаются как реальный рейтинг компании
- при появлении первого отзыва среднее рассчитывается с учётом служебных стартовых пяти звёзд

Отзывы по сделкам:
- оставляются только фактическими сторонами сделки
- проходят предварительную модерацию
- содержат оценку от одной до пяти звёзд
- содержат текстовый комментарий без медиафайлов

Получатель отзыва может:
- ответить на отзыв
- оспорить отзыв у модератора в течение одного месяца с даты публикации

Одна и та же модель рейтингов должна использоваться во всех модулях платформы.
