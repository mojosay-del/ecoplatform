## 1. Архив объявлений и предложений

### 1.1. Архив объявлений у продавца
Все объявления в финальных состояниях хранятся в архиве раздела «Торговая площадка» кабинета продавца. Архив содержит:
- объявление целиком с полным составом полей и медиафайлов на момент закрытия
- полный список поступивших предложений с их финальными статусами
- историю принятий, нажатий «Договорились» и «Не договорились»
- причину закрытия объявления

### 1.2. Доступ к архиву для покупателей
Покупатели не видят архив чужих объявлений. Покупатель видит только те объявления, по которым он подавал предложения, в разделе «Мои предложения» своего кабинета. По закрытым объявлениям покупатель сохраняет доступ к собственному предложению и его финальному статусу.

### 1.3. Переопубликация
Из архива продавец может переопубликовать объявление через кнопку «Опубликовать снова». Перед публикацией владелец может отредактировать все поля нового объявления. Переопубликация создаёт новое объявление в соответствии с разделом 7.7 (см. [listings.md](./listings.md)).

### 1.4. Неограниченное хранение
Архив хранится без ограничения по сроку в рамках первой версии. Автоочистка архива не применяется.
