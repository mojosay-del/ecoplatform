## 1. Кабинет переработчика: раздел торговой площадки

### 1.1. Структура раздела
В кабинете переработчика раздел «Торговая площадка» содержит два подраздела:
- «Лента объявлений»
- «Мои предложения»

Подраздел «Мои объявления» у переработчика в первой версии отсутствует, так как переработчик не имеет права размещать объявления о продаже в первой версии.

### 1.2. Лента объявлений у переработчика
Переработчик просматривает активные объявления заготовителей, применяет фильтры, открывает карточки объявлений и делает ценовые предложения по интересующим партиям сырья.

### 1.3. Мои предложения у переработчика
Подраздел «Мои предложения» содержит список всех предложений, поданных переработчиком, с фильтрами по статусу и сортировкой по дате.

По каждому предложению в списке отображается:
- объявление, к которому относится предложение
- статус предложения
- плашки финального состояния при их наличии
- возможность перейти в карточку объявления
- для активных предложений — возможность отредактировать или отозвать
