## 1. Тип компании: заготовитель

### 1.1. Общая характеристика
Заготовитель — основной тип компании, для которого в первой версии предусмотрен наиболее широкий рабочий кабинет.

Именно кабинет заготовителя в первой версии получает основной операционный контур платформы, включая CRM-систему и внутренние рабочие процессы.

### 1.2. Основные задачи заготовителя в системе
Заготовитель в системе должен иметь возможность:
- вести операционную работу компании
- учитывать сырьё
- работать с контрагентами
- вести сделки
- использовать CRM
- ставить задачи сотрудникам
- контролировать выполнение задач
- использовать базу знаний
- работать с торговой площадкой
- размещать объявления о продаже сырья
- продавать сырьё трейдерам и переработчикам
- отслеживать рыночные индексы и новости
- использовать калькуляторы и платные инструменты

### 1.3. Модули, доступные заготовителю
Заготовителю в первой версии доступны:
- профиль компании
- сотрудники
- роли и права доступа
- CRM-система
- база контрагентов
- сделки
- этапы и воронки
- задачи
- база знаний
- обучение
- новости
- индексы
- торговая площадка
- мои объявления
- рейтинги и отзывы
- калькуляторы
- платные инструменты
- подписка и управление доступом к модулям

### 1.4. Роли внутри кабинета заготовителя
Внутри кабинета заготовителя в первой версии могут существовать:
- собственник компании
- руководитель
- бухгалтер
- приёмщик
- менеджер по закупкам
- менеджер по продажам

Один пользователь может совмещать в себе несколько ролей, включая одновременное совмещение роли менеджера по закупкам и менеджера по продажам.

### 1.5. Особенности кабинета заготовителя
- только кабинет заготовителя имеет CRM-контур
- только кабинет заготовителя имеет воронки, этапы, базу контрагентов и задачи в рамках CRM
- только заготовитель размещает объявления о продаже на торговой площадке
